    Interface IHttpLlmFunction

    LLM function calling schema from OpenAPI operation.

    IHttpLlmFunction represents a single HTTP endpoint converted to LLM function calling format. Generated from OpenApi.IOperation as part of IHttpLlmApplication.

    Key properties:

    • name: Function name (max 64 chars for OpenAI compatibility)
    • parameters: Input schema with path/query/body merged
    • output: Response schema (undefined if void)
    • description: Critical for LLM function selection
    • validate: Built-in argument validator for error feedback

    The validate function is essential: LLMs make frequent type errors (e.g., "123" instead of 123). Validate and retry improves success rate from ~50% to 99%.
