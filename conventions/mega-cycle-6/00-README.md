## Mega-6 의 고유 임무

6 mega-cycle 의 **종결**. 일체의 오점·미해결 질문·자기모순·약한 근거를 제거하여 규약을 **최종 합의**로 확정한다. 사용자 지시 "단 하나의 오점도 없어야 하노라" 의 정신을 계승.

## Sub-1 임무

각 역할 에이전트는 **Mega-5 최종본 + 1~5 누적 cycles/ 전체** 를 읽고:

1. 누적된 모든 미해결 질문 (M1-Q* ~ M5-Q*) 이 결론을 가졌는지 점검
2. 결론 없는 질문은 이 mega 에서 명시적으로 **결정 또는 연기** (연기 시 "Phase 1 에서 결정" 등 조건)
3. 규약 ID 체계·인용·용어 통일
4. 각 규약의 위반 감지 수단 (CI 게이트·코드 리뷰 체크리스트) 이 실제로 실행 가능한지 최종 확인
