    Namespace SwaggerV2TypeChecker

    Type checker for Swagger v2.0 (OpenAPI v2) JSON schemas.

    SwaggerV2TypeChecker provides type guard functions for SwaggerV2.IJsonSchema. For typia's normalized format, use OpenApiTypeChecker instead.

    Key limitations vs OpenAPI v3.x: No oneOf/anyOf, no nullable, uses definitions instead of components.schemas, body parameters use in: "body" with schema property.
