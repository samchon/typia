    Namespace OpenApiV3

    OpenAPI v3.0 specification types.

    OpenApiV3 contains TypeScript type definitions for OpenAPI v3.0 documents. Used for parsing and generating OpenAPI v3.0 specifications. For a normalized format that unifies all OpenAPI versions, use OpenApi instead.

    Key differences from v3.1:

    • Uses nullable: true instead of type: ["string", "null"]
    • No const keyword support
    • No prefixItems for tuples
