# LLM Programmer Migration Notes

The TypeScript implementation builds LLM schemas, parameters, applications,
controllers, parsers, coercers, and structured-output bundles from metadata.
The Go backend keeps the same programmer entrypoints and emits JavaScript
source strings backed by the current JSON-schema literal helpers.

The emitted application and controller shapes retain the expected function,
validation, parse, coerce, and execution fields so transform output no longer
falls back to runtime configuration errors.
